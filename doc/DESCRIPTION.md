Materialious is a modern, privacy-friendly [Material Design](https://m3.material.io/)
front-end for YouTube, powered by [Invidious](https://invidious.io/).

This package runs Materialious in **full backend mode** (`PUBLIC_BUILD_WITH_BACKEND=true`)
with its own built-in account system backed by a PostgreSQL database, so subscriptions,
watch history and settings are stored server-side (end-to-end encrypted by the app).

Features include SponsorBlock, DeArrow, Return YouTube Dislikes, PWA support, playlists,
subscriptions and a highly customizable interface.
